# Compliance Program Assessment Checklists

## Corporate Compliance Program Elements

### DOJ Evaluation Framework

The Department of Justice evaluates compliance programs using three fundamental questions:

1. **Is the program well-designed?**
2. **Is the program being applied in good faith?**
3. **Does the program work in practice?**

## Program Design Checklist

```
COMPLIANCE GOVERNANCE ASSESSMENT
```

## Assessment Scoring

### Maturity Levels

| Level | Description | Characteristics |
|-------|-------------|-----------------|
| 1 - Initial | Ad hoc, reactive | No formal program |
| 2 - Developing | Basic elements | Policies exist, limited implementation |
| 3 - Defined | Documented program | Consistent processes, some monitoring |
| 4 - Managed | Measured and controlled | Metrics tracked, regular assessment |
| 5 - Optimizing | Continuous improvement | Best practices, predictive analytics |
