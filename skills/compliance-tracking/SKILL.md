# Compliance Tracking

This skill provides expert guidance for monitoring and managing legal and regulatory compliance obligations.

## Core Capabilities

### 1. Compliance Calendar
- Filing deadlines
- Periodic requirements
- Renewal dates
- Audit schedules

### 2. Regulatory Monitoring
- Regulatory changes
- New requirements
- Industry developments
- Enforcement trends

### 3. Compliance Reporting
- Status dashboards
- Risk assessments
- Gap analysis
- Remediation tracking

### 4. Audit Support
- Document preparation
- Audit response
- Remediation planning
- Post-audit follow-up

## Compliance Program Elements

### Program Framework
```
1. GOVERNANCE
   - Board/executive oversight
   - Compliance officer
   - Compliance committee
   - Clear accountability

2. POLICIES AND PROCEDURES
   - Code of conduct
   - Specific compliance policies
   - Procedures and controls
   - Regular updates

3. TRAINING AND COMMUNICATION
   - New hire training
   - Annual refresher training
   - Role-specific training
   - Communication of updates

4. MONITORING AND AUDITING
   - Risk assessments
   - Internal audits
   - Testing controls
   - Metrics and KPIs

5. REPORTING AND INVESTIGATION
   - Hotline/reporting channels
   - Investigation procedures
   - Documentation
   - Non-retaliation

6. ENFORCEMENT AND DISCIPLINE
   - Consistent enforcement
   - Appropriate discipline
   - Incentive integration
   - Documentation

7. RESPONSE AND REMEDIATION
   - Issue remediation
   - Root cause analysis
   - Process improvement
   - Corrective actions
```

## Regulatory Change Management

### Change Monitoring Process
1. **Identify**: Monitor regulatory sources, industry publications
2. **Assess**: Determine applicability and impact
3. **Plan**: Develop implementation approach
4. **Implement**: Execute necessary changes
5. **Document**: Record changes and compliance
6. **Monitor**: Ongoing compliance verification
