# Witness Preparation Guide

## Overview

Thorough witness preparation is essential for effective testimony. This guide covers procedures for preparing witnesses for depositions and trial testimony while maintaining ethical boundaries.

## Preparation Process

### Initial Meeting (2-3 Hours)

```
FIRST PREPARATION SESSION

Objectives:
1. Explain the deposition process
2. Review basic testimony rules
3. Assess witness demeanor and concerns
4. Begin substantive review

Agenda:

1. INTRODUCTIONS AND COMFORT (15 min)
   - Establish rapport
   - Address immediate concerns
   - Explain attorney-client privilege
   - Set expectations for preparation

2. DEPOSITION BASICS (30 min)
   - What a deposition is
   - Who will be present
   - Role of the court reporter
   - How objections work
   - Video depositions (if applicable)

3. FUNDAMENTAL RULES (45 min)
   - Listen to the entire question
   - Make sure you understand
   - Pause before answering
   - Answer only what is asked
   - Tell the truth
   - Don't guess or speculate
   - "I don't know" is acceptable
   - "I don't remember" is acceptable
   - Request breaks when needed

4. SUBSTANTIVE OVERVIEW (60 min)
   - Review key facts
   - Identify documents witness will see
   - Preview likely topic areas
   - Note areas requiring document review

5. NEXT STEPS (15 min)
   - Schedule follow-up sessions
   - Assign document review
   - Address questions
```

### Document Review Session (2-3 Hours)

```
DOCUMENT REVIEW SESSION

1. REVIEW PROTOCOL
   - Witness reviews documents before session
   - Discuss each key document
   - Confirm witness understanding
   - Note documents witness created/received
   - Identify documents witness hasn't seen

2. KEY DOCUMENTS TO REVIEW
   [ ] Documents witness authored
   [ ] Documents witness received
   [ ] Key correspondence (witness involved)
   [ ] Agreements/contracts (if relevant)
   [ ] Prior statements by witness
   [ ] Documents likely shown at deposition

3. DOCUMENT HANDLING INSTRUCTION
   - Take time to read entire document
   - Don't assume you know what it says
   - Ask if you need time to review
   - Don't explain documents you didn't create
   - Distinguish what you know vs. what document says
```

### Mock Examination (2-4 Hours)

```
MOCK DEPOSITION SESSION

1. SETUP
   - Arrange room like deposition
   - Have another attorney play opposing counsel
   - Use actual exhibits
   - Video record (optional, for review)

2. MOCK EXAMINATION
   - Cover anticipated topic areas
   - Use difficult/aggressive questioning styles
   - Include document examination
   - Test recall and consistency

3. IMMEDIATE FEEDBACK
   - Review specific answers
   - Address problematic responses
   - Reinforce good techniques
   - Practice alternative approaches

4. REPEAT DIFFICULT AREAS
   - Re-examine problem topics
   - Build confidence
   - Refine responses
```

## Core Testimony Rules

### The Ten Commandments of Testimony

```
1. TELL THE TRUTH
   - Always be truthful
   - Don't shade or spin
   - Truthful testimony protects you

2. LISTEN CAREFULLY
   - Wait for the complete question
   - Make sure you understand
   - Ask for clarification if needed

3. PAUSE BEFORE ANSWERING
   - Allows you to think
   - Allows attorney to object
   - Shows thoughtfulness

4. ANSWER ONLY WHAT IS ASKED
   - Don't volunteer information
   - Don't explain unnecessarily
   - Don't anticipate the next question

5. DON'T GUESS
   - If you don't know, say so
   - If you don't remember, say so
   - Don't estimate unless asked

6. DON'T ARGUE
   - Answer questions, don't debate
   - Stay calm and professional
   - Let the lawyer be the adversary

7. WATCH FOR COMPOUND QUESTIONS
   - Ask to have question broken down
   - Answer one thing at a time

8. READ DOCUMENTS CAREFULLY
   - Take your time
   - Read the whole document
   - Don't assume you know what it says

9. CORRECT MISTAKES
   - If you realize an error, correct it
   - Better to correct than to have inconsistency

10. STAY CALM
    - Don't get angry or defensive
    - Take breaks if needed
    - Maintain composure
```

### Common Witness Mistakes

| Mistake | Problem | Correction |
|---------|---------|------------|
| Volunteering | Provides ammunition | Stop when question answered |
| Guessing | Creates false record | Say "I don't know" |
| Arguing | Appears defensive | Answer, don't debate |
| Explaining | Invites follow-up | Answer directly |
| Assuming | May be wrong | Answer based on knowledge only |
| Rushing | Mistakes happen | Pause before answering |
| Reading documents quickly | Miss important details | Read thoroughly |
| Showing frustration | Damages credibility | Stay professional |

## Handling Difficult Questions

### Technique: "I Don't Know"

```
APPROPRIATE USES:
- You genuinely don't have the information
- The question asks about others' knowledge/intent
- The question requires speculation

HOW TO SAY IT:
"I don't know."
"I don't have that information."
"I'm not aware of that."

CAUTION:
- Must be truthful
- Will be challenged if documents show otherwise
- Don't use to avoid answering what you do know
```

### Technique: "I Don't Remember"

```
APPROPRIATE USES:
- Events from long ago
- Details not memorable
- Specific dates, times, exact words

HOW TO SAY IT:
"I don't recall."
"I don't remember specifically."
"I don't have a clear recollection."

FOLLOW-UP PREPARATION:
Q: "Would anything refresh your memory?"
A: "Perhaps if I saw a document from that time."

Q: "Did it happen or not?"
A: "I don't remember either way."
```

### Technique: Clarification

```
WHEN TO USE:
- Ambiguous questions
- Compound questions
- Questions with incorrect premises
- Unfamiliar terms

HOW TO SAY IT:
"I'm not sure I understand the question."
"Could you rephrase that?"
"What do you mean by [term]?"
"Are you asking about [X] or [Y]?"
```

### Technique: Qualifying Answers

```
APPROPRIATE QUALIFIERS:
"To the best of my recollection..."
"As I understand it..."
"Based on what I know..."
"At that time..."
"Generally speaking..."

CAUTION:
- Don't overuse
- Don't use to hedge on clear facts
- Use genuinely when appropriate
```

## Practice Scenarios

### Scenario 1: Leading Questions

```
Q: "Isn't it true that you knew about the defect before shipping?"

POOR RESPONSE:
"No, that's not true, and I resent the implication."

BETTER RESPONSE:
"No."

Q: "You never heard anyone mention a defect?"
A: "I don't recall hearing that."

Q: "So you might have heard about it?"
A: "I don't recall hearing about a defect before shipping."
```

### Scenario 2: Document Confrontation

```
Q: "I'm showing you Exhibit 5. Do you recognize this?"
A: [Take time to review] "Yes, this appears to be an email I sent."

Q: "You wrote 'We need to fix this immediately.' What did you mean?"
A: [Review document again] "I was referring to [specific issue
    mentioned in the email]."

Q: "Wasn't that about the safety problem?"
A: "The email discusses [what it actually discusses]. I don't see
    a reference to a safety problem in this email."
```

### Scenario 3: Hypotheticals

```
Q: "If I told you that three other people saw you at the meeting,
    would that refresh your memory?"

POOR RESPONSE:
"Well, if they say so, maybe I was there."

BETTER RESPONSE:
"I don't recall attending that meeting. I can only testify to
what I remember."
```

## Day-of Preparation

### Pre-Deposition Checklist

```
THE NIGHT BEFORE:
[ ] Get adequate sleep
[ ] Review key documents one more time
[ ] Review testimony rules
[ ] Prepare appropriate attire

DAY OF:
[ ] Arrive 30-60 minutes early
[ ] Meet with attorney for final prep
[ ] Review exhibits one more time
[ ] Use restroom before starting
[ ] Turn off cell phone

DURING DEPOSITION:
[ ] Maintain eye contact with questioner
[ ] Speak clearly for the record
[ ] Take offered breaks
[ ] Don't discuss testimony during breaks
[ ] Stay hydrated
```

### Attorney Support During Deposition

**Objections:**
- Attorney will object when appropriate
- Continue to wait for question after objection
- Attorney will instruct if you should not answer

**Breaks:**
- Request whenever needed
- Don't discuss testimony during breaks
- Use breaks to regroup if rattled

**Instructions Not to Answer:**
- Rare, but follow attorney's instruction
- Usually involves privilege or improper questions

## Ethical Boundaries

### What Preparation Can Include

- Explaining the deposition process
- Reviewing documents
- Discussing likely topic areas
- Practicing testimony techniques
- Conducting mock examinations
- Reviewing prior statements

### What Preparation Cannot Include

- Suggesting false testimony
- Helping witness avoid truthful answers
- Coaching specific false responses
- Encouraging witness to claim false memory loss
- Destroying or concealing relevant documents

### Key Principle

**Prepare the witness to tell the truth effectively, not to tell a false story convincingly.**
