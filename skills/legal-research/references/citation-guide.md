# Complete Legal Citation Guide

## Cases

### Federal Courts

#### Supreme Court

**Official Reporter (U.S. Reports):**
```
Brown v. Board of Education, 347 U.S. 483 (1954).
Brown v. Board of Education, 347 U.S. 483, 495 (1954). [with pincite]
```

**Parallel Citations (for older cases):**
```
Marbury v. Madison, 5 U.S. (1 Cranch) 137 (1803).
```

**Before Publication in U.S. Reports:**
```
[Case Name], 143 S. Ct. 1234 (2023).
```

#### Circuit Courts of Appeals

```
[Case Name], [Vol.] F.4th [Page] ([Circuit] [Year]).
Smith v. Jones, 45 F.4th 789, 792 (9th Cir. 2023).

Unpublished/Non-Precedential:
Smith v. Jones, No. 22-1234, 2023 WL 123456, at *3 (9th Cir. Jan. 15, 2023).
—or—
Smith v. Jones, 34 F. App'x 567 (9th Cir. 2023) (unpublished).
```

### State Courts

**General Format:**
```
[Case Name], [Vol.] [Reporter] [Page], [Pincite] ([Court] [Year]).
```

**State Supreme Courts:**
```
People v. Smith, 123 Cal. 4th 456, 460 (2023).
Smith v. State, 456 S.W.3d 789, 792 (Tex. 2023).
```

**State Intermediate Appellate Courts:**
```
Smith v. Jones, 123 Cal. App. 5th 456, 460 (2023).
Smith v. Jones, 456 N.Y.S.3d 789, 792 (App. Div. 2023).
```

**Include court identifier when not clear from reporter:**
```
Smith v. Jones, 789 A.3d 123, 125 (Pa. Super. Ct. 2023).
```

## Statutes

### United States Code

**Basic Format:**
```
[Title] U.S.C. § [Section] ([Year]).
42 U.S.C. § 1983 (2018).
```

**Multiple Sections:**
```
42 U.S.C. §§ 1983-1988 (2018). [consecutive]
42 U.S.C. §§ 1983, 1985, 1988 (2018). [non-consecutive]
```

**Subsections:**
```
26 U.S.C. § 501(c)(3) (2018).
```

**Popular Name:**
```
Americans with Disabilities Act of 1990, 42 U.S.C. §§ 12101-12213 (2018).
—on subsequent reference—
ADA
```

### State Statutes

**Official Codes:**
```
Cal. Civ. Code § 1542 (West 2023).
N.Y. Gen. Bus. Law § 349 (McKinney 2023).
Tex. Bus. & Com. Code Ann. § 17.41 (West 2023).
Del. Code Ann. tit. 8, § 102(a)(7) (2023).
735 Ill. Comp. Stat. 5/2-1001 (2023).
```

### Session Laws

**Public Laws:**
```
[Popular Name], Pub. L. No. [Number], § [Section], [Vol.] Stat. [Page] ([Year]).
Inflation Reduction Act of 2022, Pub. L. No. 117-169, § 13401, 136 Stat. 1818.
```

**State Session Laws:**
```
Act of [Date], ch. [Number], § [Section], [Year] [State] Laws [Page].
Act of June 30, 2023, ch. 123, § 5, 2023 Cal. Stat. 456.
```

## Regulations

### Code of Federal Regulations

**Basic Format:**
```
[Title] C.F.R. § [Section] ([Year]).
17 C.F.R. § 240.10b-5 (2023).
```

**Multiple Sections:**
```
40 C.F.R. §§ 260.1-.43 (2023).
```

### Federal Register

**Proposed Rules:**
```
[Title of Rule], [Vol.] Fed. Reg. [Page] (proposed [Date]) (to be codified at [C.F.R. cite]).
Climate Disclosure Rules, 87 Fed. Reg. 21,334 (proposed Apr. 11, 2022)
(to be codified at 17 C.F.R. pts. 210, 229, 232, 239, 249).
```

**Final Rules:**
```
[Title of Rule], [Vol.] Fed. Reg. [Page] ([Date]) (codified at [C.F.R. cite]).
```

### State Regulations

```
[Title], [State Admin. Code] § [Section] ([Year]).
Cal. Code Regs. tit. 2, § 18730 (2023).
N.Y. Comp. Codes R. & Regs. tit. 9, § 5.80 (2023).
```
