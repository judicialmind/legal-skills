# UTBMS Code Reference

## Overview

The Uniform Task-Based Management System (UTBMS) provides standardized codes for categorizing legal work. These codes are required by most corporate legal departments and insurance companies for e-billing.

## Usage Guidelines

1. **Always pair task and activity codes** - L110/A102 for litigation fact research
2. **Use most specific code** - L320 (Document Production) rather than L390 (Other Discovery)
3. **One task code per entry** - Block billing prohibited
4. **Expense codes standalone** - Don't combine with task codes
5. **Client-specific mapping** - Some clients require custom codes
