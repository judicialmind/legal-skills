# M&A Due Diligence Checklists by Category

## Overview

This document provides comprehensive due diligence checklists organized by functional area. These checklists serve as starting points and should be tailored based on transaction type, industry, target company characteristics, and deal-specific concerns.

---

## 1. Corporate and Organizational

### 1.1 Formation and Good Standing
- [ ] Certificate/Articles of Incorporation (and all amendments)
- [ ] Bylaws (and all amendments)
- [ ] Operating Agreement (for LLCs)
- [ ] Partnership Agreement (for partnerships)
- [ ] Good standing certificates from state of incorporation
- [ ] Good standing/qualification certificates from all states where qualified
- [ ] Foreign qualification certificates

### 1.2 Corporate Governance
- [ ] Minute books (board and shareholder meetings - 5 years)
- [ ] Written consents in lieu of meetings
- [ ] Committee charters (Audit, Compensation, Nominating)
- [ ] Committee meeting minutes
- [ ] Director and officer questionnaires
- [ ] D&O indemnification agreements
- [ ] Corporate governance guidelines
- [ ] Code of conduct/ethics policy

### 1.3 Capitalization
- [ ] Stock ledger/register of members
- [ ] Capitalization table (fully diluted)
- [ ] Stock certificates (or evidence of uncertificated shares)
- [ ] Option/warrant/convertible instruments schedule
- [ ] Equity incentive plans (all plans and amendments)
- [ ] Option grant agreements (template and exceptions)
- [ ] Restricted stock agreements
- [ ] Stock purchase agreements
- [ ] 409A valuations (last 3 years)

### 1.4 Shareholder Matters
- [ ] Shareholder agreements
- [ ] Voting agreements
- [ ] Investor rights agreements
- [ ] Registration rights agreements
- [ ] Right of first refusal agreements
- [ ] Co-sale/tag-along agreements
- [ ] Drag-along agreements
- [ ] Preemptive rights agreements
- [ ] Stockholder consents and waivers

### 1.5 Subsidiaries and Affiliates
- [ ] List of all subsidiaries (with ownership percentages)
- [ ] Organizational chart (legal entity structure)
- [ ] Intercompany agreements
- [ ] Joint venture agreements
- [ ] Partnership agreements
- [ ] Investment agreements (minority interests)
- [ ] Formation documents for each subsidiary
- [ ] Minutes for each subsidiary

## Due Diligence Best Practices

### Request List Customization
- Tailor to transaction type (asset vs. stock)
- Consider industry-specific requirements
- Focus on materiality thresholds
- Update as issues surface

### Data Room Organization
- Logical folder structure mirroring checklist
- Clear file naming conventions
- Index with descriptions
- Version control for updates

### Issue Tracking
- Log all findings
- Categorize by severity
- Assign follow-up responsibility
- Track resolution status
