# Comprehensive Federal Trial Preparation Checklist

## Overview

This checklist covers essential preparation steps for federal civil trials. Timelines reference typical scheduling order deadlines; always verify specific deadlines in the court's scheduling order and local rules.

---

## Phase 1: Pre-Trial Conference Preparation (60-90 Days Before Trial)

### Final Pretrial Conference Requirements (FRCP Rule 16)

- [ ] Review court's pretrial order requirements and local rules
- [ ] Review judge's individual practices and pretrial conference procedures
- [ ] Confer with opposing counsel regarding:
  - [ ] Stipulated facts
  - [ ] Contested issues of fact
  - [ ] Contested issues of law
  - [ ] Witness lists
  - [ ] Exhibit lists
  - [ ] Estimated trial length

### Joint Pretrial Statement/Order

Draft or contribute to joint pretrial statement including:
- [ ] Jurisdiction and venue statement
- [ ] Nature of the case
- [ ] Stipulated facts
- [ ] Contested issues of fact
- [ ] Contested issues of law
- [ ] Witnesses to be called (with summaries of expected testimony)
- [ ] Exhibits to be offered
- [ ] Pending motions
- [ ] Estimated trial length
- [ ] Settlement status
- [ ] Any special requirements (interpreter, equipment, security)

---

## Phase 2: Motions in Limine (45-60 Days Before Trial)

### Purpose
To resolve evidentiary issues before trial, preventing prejudicial evidence from reaching the jury.

### Common Motions in Limine Topics

**To Exclude:**
- [ ] Prior bad acts or character evidence (FRE 404)
- [ ] Hearsay not falling within exceptions (FRE 801-807)
- [ ] Subsequent remedial measures (FRE 407)
- [ ] Settlement negotiations and offers (FRE 408)
- [ ] Payment of medical expenses (FRE 409)
- [ ] Liability insurance (FRE 411)
- [ ] Expert testimony lacking foundation (FRE 702, *Daubert*)
- [ ] Undisclosed witnesses or exhibits
- [ ] Prejudicial photographs or videos
- [ ] "Golden Rule" arguments
- [ ] Reference to excluded claims or parties
- [ ] Mention of insurance coverage
- [ ] Improper opinion testimony (FRE 701, 702)
- [ ] Speculation about evidence not in record

**To Admit:**
- [ ] Prior inconsistent statements
- [ ] Business records (FRE 803(6))
- [ ] Public records (FRE 803(8))
- [ ] Judicial admissions
- [ ] Party opponent statements (FRE 801(d)(2))

### Motion in Limine Checklist
- [ ] Identify evidentiary issues based on discovery and pleadings
- [ ] Research applicable evidence rules and case law
- [ ] Draft motions with supporting memoranda
- [ ] File within deadline set by pretrial order
- [ ] Prepare for oral argument at pretrial conference
- [ ] Review opponent's motions in limine
- [ ] Draft oppositions to opponent's motions

---

## Phase 3: Witness Preparation (30-45 Days Before Trial)

### Witness List Management

- [ ] Compile final witness list with:
  - [ ] Full name and current contact information
  - [ ] Role in case (fact witness, expert, character)
  - [ ] Summary of expected testimony
  - [ ] Estimated time for direct and cross-examination
  - [ ] Subpoena status (if needed)
- [ ] Serve witness list per pretrial order deadline
- [ ] Review opponent's witness list

### Subpoenas (FRCP Rule 45)

**For Testimony:**
- [ ] Identify witnesses requiring subpoenas
- [ ] Issue subpoenas with proper notice (reasonable time before trial)
- [ ] Arrange witness fees and mileage
- [ ] Confirm service of subpoenas
- [ ] Calendar return dates

**For Documents (Subpoena Duces Tecum):**
- [ ] Issue subpoenas for documents from non-parties
- [ ] Allow adequate time for compliance or objection
- [ ] Review produced documents before trial

### Witness Preparation Sessions

For each witness:
- [ ] Review relevant documents witness will discuss
- [ ] Review prior deposition testimony (if applicable)
- [ ] Review discovery responses mentioning witness
- [ ] Practice direct examination questions
- [ ] Prepare for anticipated cross-examination topics
- [ ] Review courtroom procedures and expectations
- [ ] Discuss demeanor and presentation
- [ ] Address any concerns about testimony
- [ ] Confirm availability for trial dates
- [ ] Arrange travel and accommodations if needed

### Expert Witnesses

- [ ] Confirm expert report compliance with FRCP 26(a)(2)
- [ ] Review expert's deposition transcript
- [ ] Prepare demonstrative exhibits with expert
- [ ] Review expert's testimony at other trials
- [ ] Address any *Daubert* challenges
- [ ] Coordinate timing of expert's appearance
- [ ] Prepare for cross-examination on qualifications, methodology, opinions

---

## Phase 4: Exhibit Preparation (30-45 Days Before Trial)

### Exhibit List

- [ ] Compile comprehensive exhibit list including:
  - [ ] Exhibit number/letter
  - [ ] Description
  - [ ] Date
  - [ ] Author/source
  - [ ] Foundation witness
  - [ ] Objection status (stipulated/contested)
- [ ] Organize exhibits chronologically or by topic
- [ ] Serve exhibit list per pretrial order deadline
- [ ] Review opponent's exhibit list and note objections

### Exhibit Organization

- [ ] Create exhibit binders for:
  - [ ] Court (judge)
  - [ ] Witness stand
  - [ ] Counsel tables
  - [ ] Jury (if permitted)
- [ ] Tab and label all exhibits
- [ ] Include exhibit list index in each binder
- [ ] Prepare electronic versions for display

### Exhibit Foundation Requirements

Document authentication methods (FRE 901, 902):
- [ ] Testimony of witness with knowledge
- [ ] Expert testimony or comparison
- [ ] Distinctive characteristics
- [ ] Public records
- [ ] Self-authenticating documents
- [ ] Business records certification

### Demonstrative Exhibits

- [ ] Identify demonstrative exhibits needed (charts, diagrams, timelines)
- [ ] Create or commission preparation
- [ ] Verify accuracy and fairness
- [ ] Disclose to opposing counsel
- [ ] Seek advance ruling on admissibility if contested

### Pre-Marking Exhibits

- [ ] Confirm court's exhibit marking procedures
- [ ] Arrange pre-marking with clerk if available
- [ ] Use consistent marking system (Plaintiff's 1-100, Defendant's 101-200, or letters)

---

## Phase 5: Jury Instructions and Verdict Forms (21-30 Days Before Trial)

### Proposed Jury Instructions

- [ ] Review pattern jury instructions (Circuit or district-specific)
- [ ] Draft case-specific instructions
- [ ] Include legal citations and authority for each instruction
- [ ] Address all elements of claims and defenses
- [ ] Review opponent's proposed instructions
- [ ] Prepare objections to opposing instructions
- [ ] File proposed instructions by deadline

### Essential Instruction Categories

**Preliminary Instructions:**
- [ ] Role of jury
- [ ] Duty to follow law
- [ ] What is/is not evidence
- [ ] Note-taking procedures (if permitted)
- [ ] Conduct during trial

**Substantive Instructions:**
- [ ] Elements of each claim
- [ ] Elements of each affirmative defense
- [ ] Burden of proof standards
- [ ] Damages calculations
- [ ] Credibility determinations

**Closing Instructions:**
- [ ] Deliberation procedures
- [ ] Unanimity requirements
- [ ] Verdict form completion
- [ ] Foreperson selection

### Verdict Form

- [ ] Draft proposed verdict form(s)
- [ ] Ensure form addresses all claims and defenses
- [ ] Include damages categories as appropriate
- [ ] Consider special interrogatories vs. general verdict
- [ ] Confer with opponent regarding agreed form
- [ ] File proposed verdict form by deadline

---

## Phase 6: Trial Briefs and Legal Preparation (14-21 Days Before Trial)

### Trial Brief

- [ ] Summarize key facts
- [ ] Outline legal theories and elements
- [ ] Address anticipated evidentiary issues
- [ ] Distinguish adverse authority
- [ ] Include copies of key cases as appendix

### Legal Research Updates

- [ ] Check for new relevant case law since dispositive motions
- [ ] Update statutory citations
- [ ] Research any novel issues raised in motions in limine
- [ ] Prepare pocket briefs on anticipated evidentiary issues

### Anticipated Legal Issues

- [ ] Prepare argument on burden of proof
- [ ] Research any Statute of Frauds issues
- [ ] Address statute of limitations if contested
- [ ] Prepare damages calculation authority

---

## Phase 7: Final Trial Preparation (7-14 Days Before Trial)

### Opening Statement

- [ ] Outline key themes
- [ ] Draft narrative structure
- [ ] Identify critical evidence to reference
- [ ] Practice delivery and timing
- [ ] Prepare demonstrative exhibits for opening
- [ ] Do NOT argue the case (save for closing)

### Direct Examination Outlines

For each witness:
- [ ] Write question-by-question outline
- [ ] Incorporate exhibit foundations
- [ ] Plan order of witnesses
- [ ] Anticipate objections and prepare responses
- [ ] Prepare redirect questions for likely cross-examination points

### Cross-Examination Preparation

For each opposing witness:
- [ ] Review deposition transcripts
- [ ] Identify impeachment opportunities
- [ ] Prepare confrontation exhibits (prior inconsistent statements)
- [ ] List key admissions to obtain
- [ ] Practice tone and technique

### Closing Argument

- [ ] Outline structure around key themes
- [ ] Integrate admitted evidence and testimony
- [ ] Address jury instructions
- [ ] Prepare rebuttal points
- [ ] Note: Cannot be finalized until evidence closes

### Courtroom Logistics

- [ ] Visit courtroom if possible
- [ ] Note location of podium, witness stand, exhibits
- [ ] Confirm technology setup (projector, document camera, ELMO)
- [ ] Test all equipment
- [ ] Coordinate with courtroom deputy
- [ ] Confirm court reporter availability

---

## Phase 8: Trial Week Preparation (Week Before Trial)

### Administrative Matters

- [ ] Confirm trial date and time with clerk
- [ ] Confirm daily schedule (start/end times, breaks)
- [ ] Arrange parking and access
- [ ] Prepare daily trial folders
- [ ] Organize war room or breakout space

### Witness Coordination

- [ ] Confirm all witnesses' availability
- [ ] Provide witnesses with trial schedule
- [ ] Arrange for witnesses to be on-call
- [ ] Finalize travel and lodging arrangements

### Equipment and Supplies

- [ ] Trial bags/boxes with supplies
- [ ] Extra copies of all exhibits
- [ ] Laptop, charger, extension cords
- [ ] Document camera familiarity
- [ ] Presentation software ready
- [ ] Note pads, pens, highlighters
- [ ] Post-it notes, tabs, clips
- [ ] Portable printer (if needed)
- [ ] Daily transcript orders (if desired)

### Client Preparation

- [ ] Final trial preparation meeting with client
- [ ] Review courtroom decorum
- [ ] Discuss appropriate attire
- [ ] Explain trial procedures and timeline
- [ ] Discuss potential outcomes
- [ ] Review testimony if client will testify
- [ ] Discuss settlement authority

---

## Phase 9: During Trial

### Daily Procedures

- [ ] Review prior day's transcript (if ordered)
- [ ] Adjust witness order as needed
- [ ] Update trial outline based on developments
- [ ] Confer with client on developments
- [ ] Track admitted vs. excluded exhibits
- [ ] Note jury reactions and demeanor
- [ ] Preserve objections for appeal

### Preserving the Record

- [ ] Make timely and specific objections
- [ ] Request sidebar for sensitive matters
- [ ] Make offers of proof for excluded evidence
- [ ] Request limiting instructions where appropriate
- [ ] Move for judgment as matter of law (FRCP 50) at close of opponent's case

### Jury Management

- [ ] Observe juror engagement
- [ ] Note potential issues (sleeping, smartphones)
- [ ] Alert court to any juror issues
- [ ] Do not communicate with jurors

---

## Phase 10: Post-Trial Matters

### Immediately After Verdict

- [ ] Poll jury if desired
- [ ] Request jurors be available for post-verdict discussion (if permitted)
- [ ] Note any verdict form irregularities

### Post-Trial Motions (Timeline Critical)

**FRCP Rule 50(b) - Renewed Motion for Judgment as Matter of Law:**
- Must be filed within 28 days after entry of judgment
- Prerequisite: Must have moved under Rule 50(a) at close of evidence

**FRCP Rule 59 - Motion for New Trial:**
- Must be filed within 28 days after entry of judgment
- Grounds: verdict against weight of evidence, erroneous jury instructions, improper conduct, newly discovered evidence, excessive/inadequate damages

**FRCP Rule 59(e) - Motion to Alter or Amend Judgment:**
- Must be filed within 28 days after entry of judgment

### Appeal Considerations

- [ ] Note deadline for notice of appeal (60 days if government party; 30 days otherwise)
- [ ] Identify preserved issues for appeal
- [ ] Order trial transcript
- [ ] File notice of appeal if appropriate
